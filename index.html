Tentang Saya

Seorang fresh graduate Teknik Informatika yang sedang mendalami dunia copywriting, desain UX, dan content writing.

Saat ini, saya fokus membangun personal branding di media sosial seperti TikTok dan aktif membuat konten yang relevan. Saya juga rutin berbagi tulisan di blog pribadi melalui platform seperti Medium. Selain belajar, saya senang mengerjakan proyek pribadi atau membuat studi kasus untuk mengasah keterampilan dan memahami lebih dalam bidang yang saya tekuni. Saya terus mengembangkan wawasan dengan memanfaatkan berbagai sumber gratis, seperti YouTube, kursus daring gratis, dan platform lainnya.

Proyek

Proyek Pribadi

Beberapa proyek di bawah ini saya kerjakan sebagai bagian dari tugas perkuliahan maupun proyek pribadi. Beberapa di antaranya saya unggah di akun Behance, dan Medium sebagai dokumentasi.

UI Design Aplikasi NutriBalance

NutriBalance adalah proyek akhir yang saya kerjakan bersama tim dalam Program Bangkit Academy 2023. Saya ikut bertanggung jawab mendesain versi alternatif aplikasi ini menggunakan Figma untuk meningkatkan pengalaman pengguna.

E-Commerce Website

Proyek ini merupakan tugas mata kuliah Pemrograman Berorientasi Objek (PBO) yang saya kerjakan dengan menggunakan konsep-konsep OOP serta bantuan library Bootstrap. Website ini merupakan platform jual sepatu online, di mana saya mengimplementasikan struktur kelas dan objek untuk mengatur data produk, transaksi, dan user experience.

Re-Design Website

Saya mencoba mendesain ulang website e-commerce yang menjual tas. Dalam prosesnya, saya memanfaatkan warna dasar produk sebagai inspirasi utama dan mengatur ulang tata letak elemen untuk menciptakan tampilan yang lebih menarik. Saya juga menambahkan beberapa fitur baru untuk meningkatkan kemudahan penggunaan website bagi pengguna.

DeliveryFood App

Proyek ini merupakan proyek eksperimen yang terinspirasi dari aplikasi pemesanan makanan populer di Indonesia.
